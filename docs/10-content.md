## 7.8 Чек-лист контента

### Обязательные элементы на каждой странице:

- [ ] Уникальный title (50-60 символов)
- [ ] Уникальный meta description (150-160 символов)
- [ ] H1 заголовок
- [ ] Структурированный контент (H2-H6)
- [ ] Альты у всех изображений
- [ ] Внутренние ссылки

### Для услуг:

- [ ] Название (понятное и с ключом)
- [ ] Краткое описание (для списка)
- [ ] Полное описание (минимум 500 слов)
- [ ] Цена или диапазон цен
- [ ] Сроки выполнения
- [ ] Преимущества / особенности
- [ ] Галерея (3-5 фото)
- [ ] Схожие услуги

### Для портфолио:

- [ ] Название проекта
- [ ] Описание (что делали)
- [ ] Фото "до" и "после"
- [ ] Галерея процесса
- [ ] Информация о клиенте
- [ ] Площадь и сроки
- [ ] Ссылка на услугу

### Для отзывов:

- [ ] Имя автора
- [ ] Рейтинг (1-5 звезд)
- [ ] Текст отзыва
- [ ] Привязка к услуге/проекту
- [ ] Метка "Проверен" (если реальный клиент)
