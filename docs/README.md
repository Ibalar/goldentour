# «Золотой Тур» — строительная компания

Современный корпоративный сайт для строительной компании, включающий каталог услуг, портфолио выполненных работ, калькулятор стоимости и систему управления контентом.

## Описание проекта

Сайт разработан для презентации услуг строительной компании «Золотой Тур», демонстрации портфолио объектов и привлечения новых клиентов через встроенные формы заявок и калькулятор.

### Основные возможности

- **Каталог услуг** — детальное описание строительных и отделочных работ с ценами
- **Портфолио** — галерея выполненных объектов с фильтрацией по категориям
- **Онлайн-калькулятор** — расчёт примерной стоимости работ
- **Формы заявок** — удобный способ связи с компанией
- **Отзывы клиентов** — система отзывов с модерацией
- **Административная панель** — полное управление контентом

## Технологический стек

| Компонент | Технология |
|-----------|------------|
| Backend | Laravel 11 (PHP 8.3+) |
| Админ-панель | MoonShine 4 |
| Frontend | Blade Templates |
| Стили | Tailwind CSS 3 |
| JavaScript | Alpine.js |
| База данных | MySQL 8.0+ |
| Сборка | Vite |
| Сервер | Nginx |
